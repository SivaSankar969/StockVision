# StockVision

StockVision is an interactive dashboard built with Python and Streamlit that uses machine learning to analyze and predict stock market trends. It helps investors make data-driven decisions with visualizations, forecasts, and performance metrics.

## Features
- Interactive data visualization for historical stock performance
- Stock price prediction using machine learning models (Linear Regression)
- Real-time interactive interface with Streamlit
- Evaluation of model performance using MAE, MSE, and RÂ² scores

## Technologies Used
- Python
- Scikit-learn
- Pandas, NumPy
- Matplotlib, Seaborn, Plotly
- Streamlit
